Introducción

Desde los orígenes del Internet, HTML (Hypertext Markup Language) se ha posicionado como el lenguaje estándar para la estructuración, la conexión y la puesta a disposición, independientemente de la plataforma, de contenidos en la World Wide Web. Con el lenguaje HTML los programadores de páginas web pueden trabajar de manera totalmente independiente para desarrollar páginas web, las cuales tienen las características de ser seguras y accesibles para el usuario, quien accede a la página por medio de un navegador web, y se le muestran los contenidos sin importar el navegador que utilice. La siguiente investigación contiene información sobre el lenguaje de marcado HTML, en el que, mediante el estudio de los antecedentes de este tema, se lograron identificar los objetivos que se desean alcanzar así como dar a conocer el tema a través de un contenido lleno de información y de fácil comprensión, que genere un impacto positivo en la persona que lea y analice el documento, y la nutra de conocimientos acerca del lenguaje HTML, un lenguaje que va cobrando cada vez más relevancia en el mundo moderno, y no deja de evolucionar.



Antecedentes

El lenguaje de marcado HTML ha pasado por un largo proceso de evolución, y todavía no ha terminado de evolucionar, ya que nuevas funciones y etiquetas se añaden constantemente. Un trabajo que muestra esta evolución es del autor Hugo Delgado (2019), cuyo título es “Historia de HTML”. El trabajo incluye una descripción de cómo HTML ha ido evolucionando a través de los años, desde su origen en 1980, propuesto por Tim Berners-Lee como un sistema de hipertexto que permitía compartir documentos, y todos los nuevos estándares o versiones que han surgido a lo largo del tiempo. Otro trabajo sobre este tema realizado por Juan López Ricourt (2016) llamado “La Evolución del HTML”, aporta mucho, pues expone la estructura básica de una página web programada con HTML, así como la importancia del lenguaje de estilos CSS e incluye una lista con varias etiquetas utilizadas para diferentes cosas en el lenguaje HTML. Un trabajo más realizado por Elisabet Rodríguez (2006), con título “El HTML, una idea en evolución”, enfatiza en la creación y las personas involucradas en el desarrollo del lenguaje. En el trabajo se exhibe de forma detallada el proceso de creación de HTML por Berners Lee, quien ideó HTML en su época de trabajo en el CERN (Centro Europeo para la investigación Nuclear) con el fin de hacer más sencillo el acceso a la información, y también identifica a Ted Nelson como el primero en utilizar el término hipertexto, y lo utilizaba para referirse a escritos no secuenciales. Como última investigación está la obra de Alfredo Ruiz (2014) titulada “¿Cuáles son las características principales del HTML?”. Se trata de un trabajo útil para todos aquellos principiantes en este ámbito, pues indica lo que tiene que tener cada división de una página web, así como indica cuáles son las etiquetas indispensables y las reglas “gramaticales” para el lenguaje de marcado HTML.



Justificación

La presente investigación se realiza con el fin de informar a las personas acerca del lenguaje de marcado HTML, que las personas sepan para qué sirve este lenguaje, que estén al tanto del origen y de los usos que el lenguaje HTML tiene, ya que actualmente la creación de páginas web es muy importante en los campos laborales, y tener conocimientos sobre HTML permite acceder a oportunidades. El trabajo resuelve las incógnitas que podrían tener algunas personas, como para qué sirve HTML, cuáles han sido todas sus versiones, cuál fue su origen, cuáles son las etiquetas básicas que hay que saber al introducirse en la programación HTML, entre otras. Además, mediante la creación de una página web, plasmaré todo lo que he podido aprender sobre el lenguaje HTML, demostrando cómo se usa y qué cosas se pueden crear utilizando este lenguaje.



Objetivos

1. Definir qué es el lenguaje HTML: Es fundamental definir exactamente qué es el lenguaje HTML, para que el lector pueda entender a la perfección el tema central del proyecto.


2. Dar a conocer la importancia del lenguaje de marcado HTML en la vida actual: El lenguaje HTML es un lenguaje que se usa para la creación de las páginas web, un recurso que en la actualidad es ampliamente utilizado por las empresas para crear sitios de venta de productos, informativos, recreativos, entre otros. Es por eso que saber utilizar el lenguaje HTML abre muchas puertas en el ámbito laboral.


3. Poner en práctica todos los conocimientos que he adquirido en la realización de la página web: Lo que he aprendido en los últimos años sobre el lenguaje HTML me permitirá realizar una página web, que contendrá toda la información mostrada en este proyecto.


4. Describir la evolución del lenguaje HTML a través de los años: Como todo lenguaje de programación, HTML ha ido cambiando, y varias versiones de HTML han salido, las cuales se describirán en este trabajo.
